 # nico酒店 デモサイト

## 概要
本リポジトリは、「ニコ酒店」のために作成したデモWebサイトです。  
お店の世界観や雰囲気を伝えるシンプルで落ち着いた構成を目指し、HTML/CSS/JavaScriptの基礎スキルを活かして制作しました。

## 制作の背景と目的
- JavaScript学習課題「アコーディオンメニュー」を応用し、実案件を想定した応用演習として制作
- 「nico酒店」の店舗オーナーに提案予定のデモページとして、案件獲得を目指した実践練習
- プログラミングの勉強を一から始めて約50日間の学習成果の総まとめとして、Web制作スキルの定着を目的に作成

## 使用技術
- HTML5（セマンティックタグを意識した構成）
- CSS3（Flexbox / Grid / レスポンシブ対応 / カスタムデザイン）
- JavaScript（アコーディオンメニュー、Swiper.jsによるスライダー機能）
- メディアクエリによるモバイル最適化（スマホ/タブレット/PC）

## 主な機能
- 商品スライダー（Swiper.js使用）
- レスポンシブレイアウト対応
- シンプルなハンバーガーメニュー（JavaScript制御）
- お店紹介／地図埋め込み／Instagramリンクボタン　など

## 今後の展望

- ポートフォリオに掲載し、学習と実践の成果物として活用

#### 学び・気づき

- **Swiper.jsのレスポンシブ設定**
  - `.swiper-slide` に `width: 100% !important;` を設定することで、スマホで1枚表示に制御できる
  - `.swiper-pagination` を `position: static` にして、カード下部に独立表示できるようにした

- **文章途中での文字折り返し設定の使い方**
  - `word-break`：単語や文字を途中で強制的に折り返す  
    - 例：`word-break: keep-all;` → 日本語や単語の途中で折り返さない（自然な改行）
  - `overflow-wrap`：長すぎる単語を折り返す  
    - 例：`overflow-wrap: break-word;` → 幅を超える単語を折り返してレイアウト崩れ防止
  - `white-space`：テキストの空白や折り返し方法を設定  
    - 例：`white-space: normal;` → 通常の折り返しルールに戻す

- **画像とテキストの比率調整**
  - `.store-concept-container` の左右にあるテキストと画像の `flex` 値を調整してバランスを取った  
    - 例：`flex: 1`（テキスト） / `flex: 1.2`（画像）などで調整
  - `aspect-ratio` を使って画像の縦横比を統一  
    - 例：`aspect-ratio: 4 / 3;`

- **各セクションの最大幅制御**
  - `.container` に `max-width: 1200px;` を設定  
    - → 画面全体が広すぎるとテキストが読みづらくなるため、読みやすい幅に制限
    - → フルHDやMacのフルスクリーン時でもバランスよく見える

- **お茶商品カードの微調整**
  - `.tea-item h3` の文字サイズや中央寄せバランスを調整
  - `.tea-price` 内で価格と「税込」表示に差をつけるために `span` を使ってフォントサイズを個別指定

    ```html
    <p class="tea-price">¥750 <span class="tax">（税込）</span></p>
    ```

    ```css
    .tea-price .tax {
      font-size: 0.8rem;
    }
    ```
