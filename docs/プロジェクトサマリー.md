# プロジェクトサマリー

## 1. プロジェクトのゴール

ニコ酒店の公式ウェブサイトを構築し、オンラインでの情報発信とブランドイメージ向上を図る。将来的にはEC機能の実装も視野に入れる。

## 2. 現在のフェーズ

静的デモサイト（HTML/CSS/JavaScript）の制作が完了し、クライアントからの承認を得た段階。
これより、本番実装フェーズへ移行する。

## 3. 完成済みの成果物

- **静的デモサイト**
  - `index.html` および `site/` ディレクトリ配下の関連ファイル群

- **各種ドキュメント**
  - **要件定義書:** `docs/proposals/0906/要件定義兼見積書.md`
  - **サイト仕様書:** `docs/spec/` ディレクトリ配下のファイル群
  - **その他:** `docs/` 配下に格納されている議事録や提案資料

## 4. 決定済みの技術スタック

- **フロントエンドフレームワーク:** Next.js
- **ホスティング環境:** 提案済み (`docs/proposals/0906/ホスティングサービス提案書.pdf`)。最終決定は別途行う。

## 5. 今後の短期的なタスク（Next Step）

1. **リポジトリの分離:**
   - 本番実装用の新しいGitリポジトリを作成する。
   - 現行リポジトリはアーカイブとして保存する。
   - （詳細は `docs/repository_management_plan.md` を参照）

2. **プロジェクトの初期セットアップ:**
   - 新規リポジトリにNext.jsプロジェクトをセットアップする。

3. **ドキュメントと資産の移行:**
   - `spec`関連のドキュメントや、`site/images` 内の画像アセットなどを新規リポジトリに移行する。

4. **開発開始:**
   - 静的デモを参考に、Next.jsの作法に沿ってコンポーネントの作成を開始する。
