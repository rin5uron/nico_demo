# 参考サイト分析とデザインの方向性

## 1. 参考サイト

- **系列サイト:**
    - [深川蒸留所](https://fukagawa-distillery.tokyo/)
        - デザインテイストの基盤となるサイト。黒と白を基調とした、ミニマルで洗練されたデザイン。

- **方向性の参考:**
    - [虎ノ門蒸溜所](https://toranomondistillery.jp/)
        - 同じ系統
    - [八海山](https://haccaisan.com/)
        - 余白を活かしたレイアウトと、写真・タイポグラフィの美しさが参考になる。
    - [新澤醸造店](https://www.niizawa-sake.jp/)
        - 黒を基調とした高級感のあるデザインと、製品が際立つ見せ方が参考になる。

## 2. デザインの方向性

系列サイトである「深川蒸留所」との統一感を重視しつつ、ニコ酒店ならではの個性を表現する。

- **基本テイスト:**
    - 深川蒸留所のサイトが持つ、**ミニマルで洗練された雰囲気**を継承する。
    - 八海山や新澤醸造店のように、**余白を活かし、写真とタイポグラフィで魅せる**デザインを目指す。

- **配色:**
    - 深川蒸留所のモノクロ基調とは差別化を図る。
    - 名刺のデザインを元に、**深緑 (`#336A4A`)** と **生成り色 (`#F8F5F0`)** をメインカラーに据え、ニコ酒店独自の温かみやストーリー性を表現する。

- **フォント:**
    - 深川蒸留所と同様に、**上品な明朝体**（游明朝体、筑紫明朝など）をメインに使用し、グループとしての統一感を出す。
    - 本文には可読性の高いモダンなゴシック体（サンセリフ体）を組み合わせる。

- **レイアウト:**
    - 情報を詰め込みすぎず、**シンプルで分かりやすい構成**を最優先とする。
    - 商品の写真や、店舗の雰囲気が伝わる写真を大きく扱い、視覚的にブランドイメージを訴求する。
