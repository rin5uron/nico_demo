# サイトコンセプト（仮）

## 1. ターゲット顧客

- **年齢層:** 30代〜50代
- **性別:** 男女問わず
- **ライフスタイル:**
    - 食や暮らしにこだわりを持ち、質の良いものを求める方
    - 作り手の想いやストーリーに共感し、丁寧なものづくりを好む方
    - 特別な日のためのお酒や、大切な人への贈り物としてお酒を選ぶ方
    - 地域（門前仲町・清澄白河エリア）の文化やお店に興味がある方

## 2. サイトのゴール

- **短期的なゴール:**
    現在、Instagramと準備中のオンラインショップ(BASE)に点在している情報を集約し、顧客がスムーズに情報にアクセスできる公式のハブサイトを構築する。まずは「ニコ酒店」の公式サイトとして、ブランドの信頼性と認知度を高める。
- **長期的なゴール:**
    系列店であるバー、カレー屋、酒店を統合する「ニコグループ」のポータルサイトを構築する。各店舗の情報を発信し、相互送客を促進することで、グループ全体のブランド価値を高め、ファンを増やす。

## 3. サイトコンセプト（仮）

**「一杯に、物語を。一皿に、想いを。」**

ただお酒を売る場所ではなく、作り手の情熱や、その土地の風土、お酒が生まれるまでの物語を伝えるサイト。
訪れた人が、次の一杯を選ぶのがもっと楽しくなるような、発見と体験を提供します。

## 4. デザインの方向性

### 配色
- **メインカラー:** 深みのある緑 (`#336A4A` / 名刺の背景色)
- **ベースカラー:** オフホワイト・生成り色 (`#F8F5F0` / 名刺の紙の色)
- **アクセントカラー:** ロゴに使われている、より濃い緑 (`#2A4B3A`あたり)

### フォント
- **日本語:** 游明朝体、筑紫明朝など、上品で可読性の高い明朝体
- **英語:** Helvetica Neue、Interなど、シンプルでモダンなサンセリフ体

### デザインキーワード
- **メインキーワード:**
    - 上質、洗練、モダン、静寂、余白
- **サブキーワード:**
    - 温かみ、手仕事感、 authenticity（本物であること）、ストーリー性
