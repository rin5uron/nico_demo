# サイトマップ

## 1. サイト構成

お急ぎのサンプル作成、および短期的なゴールである「情報集約」を最優先とし、まずは単一のトップページ内に各コンテンツを配置する**1ページ構成**とする。
将来的には、各セクションを独立したページとして拡張することも見据えた設計とする。

## 2. ページ内コンテンツ構成

- **トップページ (`/`)**
    - **ヘッダー**
        - ロゴ
        - グローバルナビゲーション
            - お店について (セクション2へのアンカーリンク)
            - 取扱商品 (セクション3へのアンカーリンク)
            - お知らせ (セクション4へのアンカーリンク)
            - イベント (セクション5へのアンカーリンク)
            - お問い合わせ (セクション6へのアンカーリンク)
        - SNSリンク (Instagram)

    - **セクション1：メインビジュアル**
        - サイトの顔となるキービジュアルとキャッチコピーを配置。

    - **セクション2：お店について (About)**
        - 店舗のコンセプト、歴史、こだわりを伝えるテキストと写真を配置。

    - **セクション3：取扱商品 (Products)**
        - **お酒:** スライダー形式で商品写真と簡単な説明を掲載。オンラインショップへのリンクを設置。
        - **薬膳茶:** 4商品を一覧で紹介。写真、商品名、説明、価格を掲載。

    - **セクション4：お知らせ (News)**
        - Instagramのフィードをサイト内に埋め込み、リアルタイムな情報を表示。
        - CMSの導入は将来的な課題とし、まずは手動更新またはInstagram埋め込みで対応。

    - **セクション5：イベント (Events)**
        - 直近のイベント情報を掲載するエリア。詳細や過去の様子はInstagramへのリンクで補完。

    - **セクション6：お問い合わせ (Contact)**
        - 店舗情報（住所、地図）を掲載。
        - 「お問い合わせはInstagramのDMまで」と明記。

    - **フッター**
        - コピーライト
