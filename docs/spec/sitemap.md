# サイトマップ

## 1. サイト構成

お急ぎのサンプル作成、および短期的なゴールである「情報集約」を最優先とし、まずは単一のトップページ内に各コンテンツを配置する**1ページ構成**とする。
将来的には、各セクションを独立したページとして拡張することも見据えた設計とする。

## 2. ページ内コンテンツ構成

- **トップページ (`/`)**
    - **ヘッダー**
        - ロゴ
        - グローバルナビゲーション
            - お知らせ (セクション2へのアンカーリンク)
            - イベント (セクション3へのアンカーリンク)
            - お店について (セクション4へのアンカーリンク)
            - 取扱商品 (セクション5へのアンカーリンク)
            - お問い合わせ (セクション6へのアンカーリンク)
        - SNSリンク (Instagram)

    - **セクション1：メインビジュアル**
        - サイトの顔となるキービジュアルとキャッチコピーを配置。

    - **セクション2：カレンダー・最新情報 (Calendar & Latest Info)**
        - **枠外テキスト:** 「あ」の予測変換テキストをセクションタイトル直下に配置し、ページの幅いっぱいに表示。
        - **コンテンツエリア:** デスクトップでは左側に営業カレンダー画像、右側にInstagramの最新投稿プレビューを並列配置し、高さを揃える。
        - モバイルでは縦並びで表示。

    - **セクション3：お知らせ（ニュース一覧）(News List)**
        - 最新のお知らせタイトルを数件羅列し、「一覧へ」リンクで詳細ページへ誘導。（現時点ではリンク先は未定）

    - **セクション4：イベント (Events)**
        - 直近のイベント情報を掲載するエリア。詳細や過去の様子はInstagramへのリンクで補完。

    - **セクション5：お店について (About)**
        - 店舗のコンセプト、歴史、こだわりを伝えるテキストと写真を配置。

    - **セクション6：取扱商品 (Products)**
        - **お酒:** スライダー形式で商品写真と簡単な説明を掲載。オンラインショップへのリンクを設置。
        - **薬膳茶:** 4商品を一覧で紹介。写真、商品名、説明、価格を掲載。

    - **セクション7：お問い合わせ (Contact)**
        - 店舗情報（住所、地図）を掲載。
        - 「お問い合わせはInstagramのDMまで」と明記。

    - **フッター**
        - コピーライト
