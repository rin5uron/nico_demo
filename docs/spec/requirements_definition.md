# 要件定義書

## 1. プロジェクト概要

ニコ酒店様の公式Webサイトを制作し、ブランドイメージの向上と、商品の魅力を伝えることを目的とする。
シンプルかつ洗練されたデザインで、作り手の想いや商品のストーリーを伝えることに重点を置く。

## 2. 要件一覧

| ID | 要件 | 種別 | 詳細・目的 | 優先度 | 備考 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **【デザイン要件】** | | | | | |
| D-01 | 全体デザイン | デザイン | シンプルで洗練された、上質でモダンなデザイン。余白を活かし、作り手の想いが伝わるような雰囲気。 | 高 | 白ベース、無彩色に近い配色。 |
| D-02 | ロゴ | デザイン | 既存のロゴを使用。サイトのデザインに合わせて最適化する。 | 高 | 高解像度のデータをご提供いただく。 |
| D-03 | 写真 | デザイン | 商品や店舗の写真をメインに、ブランドイメージを視覚的に伝える。 | 高 | プロのカメラマンによる撮影を推奨。 |
| **【コンテンツ要件】** | | | | | |
| C-01 | トップページ | コンテンツ | サイトの顔。ブランドイメージと主要コンテンツへの導線を配置。 | 高 | メインビジュアルが重要。 |
| C-02 | ニコ酒店について | コンテンツ | 店舗のコンセプト、歴史、こだわりなどを伝える。 | 高 | 原稿と写真をご提供いただく。 |
| C-03 | 取扱商品ページ（お茶） | コンテンツ | 4つの商品（雪中、緑陰、暗香、草萌）を一覧で表示する。 | 高 | 写真と商品名を中心に、説明・価格を表示。 |
| C-04 | 取扱商品ページ（お酒） | コンテンツ | 商品写真、商品名、一言を紹介。詳細はオンラインショップ（外部）へ誘導する。 | 高 | 現状はサンプル表示（スライダー／ギャラリー等）。正式運用時はBASE等への直リンクに切替。 |
| C-05 | お知らせ/ブログ | コンテンツ | 新商品やイベント、季節の便りなどを発信する。 | 中 | 現状は静的リストでサンプル表示。運用時の更新方法は要検討。 |
| C-06 | イベント情報 | コンテンツ | 定期イベントの紹介、ギャラリー。 | 中 | 現状は静的メッセージ＋Instagramへの導線。運用方法は要検討。 |
| C-07 | お問い合わせ | コンテンツ | 店舗情報、アクセス、「お問い合わせはInstagramのDMまで」と記載。 | 高 | 地図の埋め込みを含む。 |
| C-08 | ナビゲーション | コンテンツ | ヘッダーナビに「お知らせ／お店について／取扱商品／お問い合わせ」を配置。モバイルはハンバーガーで開閉。 | 高 | イベントはナビに含めない。右側にInstagram／オンラインショップの外部リンクアイコンを配置。 |
| **【機能要件】** | | | | | |
| F-01 | レスポンシブ対応 | 機能 | スマートフォン、タブレット、PCなど、どのデバイスでも最適に表示される。 | 高 | | 
| F-02 | Instagram連携 | 機能 | Instagramの投稿をサイト内に埋め込み、リアルタイムな情報を発信する。 | 低 | 現状はプレースホルダ＋リンク表示。`embed.js`は無効化中（後日有効化可能）。 |
| F-03 | CMS（コンテンツ管理） | 機能 | お知らせ/インスタグラムの更新を随時追加できるようにする。 | 中 | 使用コンテンツ等要検討、要件定義決定まではサンプルデザインのみ作成予定。 |
