# 要件定義書

## 1. プロジェクト概要

ニコ酒店様の公式Webサイトを制作し、ブランドイメージの向上と、商品の魅力を伝えることを目的とする。
シンプルかつ洗練されたデザインで、作り手の想いや商品のストーリーを伝えることに重点を置く。

## 2. 要件一覧
