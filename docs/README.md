# `docs` ディレクトリ運用ルール

このディレクトリは、ニコ酒店様Webサイト制作プロジェクトに関するすべてのドキュメントを管理する場所です。
情報は以下のルールに従って、各ディレクトリに格納してください。

---

## ディレクトリ構成

- ### `contacts/`
  - **目的:** 顧客とのやり取りの記録を保管します。
  - **主な内容:**
    - ヒアリング議事録
    - メールでのやり取りの控え
    - 定例会の議事録

- ### `media/`
  - **目的:** この `docs` ディレクトリ内で使用する画像などのメディアファイルを保管します。
  - **主な内容:**
    - ドキュメントに貼り付けるスクリーンショット
    - 図解用の画像

- ### `notes/`
  - **目的:** プロジェクトを進める上での個人的なメモや、調査記録、学習内容を自由に書き留める場所です。
  - **主な内容:**
    - 技術調査のメモ
    - 参考にした記事のURL
    - アイデアの断片

- ### `proposals/`
  - **目的:** 顧客への提出書類や、契約関連のファイルを保管します。
  - **主な内容:**
    - 提案書
    - 見積書
    - 請求書
    - 契約書

- ### `spec/`
  - **目的:** Webサイトの仕様を定義するドキュメントを保管します。
  - **主な内容:**
    - 要件定義書
    - サイトマップ
    - ワイヤーフレーム
    - 機能一覧
    - デザイン仕様書
