# 提案要約（ニコ酒店様）

- **目的**: Instagram と オンラインショップ（BASE 等）に散らばる情報を、公式サイトで「見やすく一元化」
- **ゴール**
  - **短期**: 信頼性と認知を高める 1 ページ構成のハブサイト
  - **長期**: グループ全体（ニコ酒店 / バー / カレー）のポータルへ拡張可能
- **対象ユーザー**: 30–50 代、質にこだわる / ストーリーに共感する層、地域に関心のある方
- **サイト構成（1 ページ内）**
  - メインビジュアル
  - カレンダー + Instagram（最新情報）
  - お知らせ（静的リスト）
  - イベント（Instagram への導線）
  - お店について（写真 + 紹介文）
  - 取扱商品（お茶: 4 商品 / お酒: 外部ショップ誘導）
  - お問い合わせ（住所・地図・Instagram DM）
- **デザイン方向**: 深緑 × 生成り、上品な明朝体。参考: 深川蒸留所 / 虎ノ門蒸溜所
- **実装範囲**
  - レスポンシブ（スマホ / タブレット / PC）
  - Instagram 枠は現状プレースホルダ表示（後日 embed.js を有効化可）
  - 写真中心の見せ方、軽快な表示、アクセシビリティ配慮
- **成果物**: トップページ一式、公開手順メモ
- **スケジュール感**: 要件確定後 約 2 週間で実装 → 確認 → 公開
- **前提 / 除外**
  - 写真・ロゴ・紹介文のご提供
  - CMS 導入は後日検討（今回の範囲外）
- **要確認事項（打合せで決定）**
  - ナビ文言の最終確定、年齢確認の文言 / 位置
  - Instagram 埋め込みの運用方針（有効化時期）
  - 画像点数とテキスト分量
  - ドメイン / 公開方法（Vercel 可）
- **次のアクション**
  1. 本要約の合意
  2. 打合せで詳細すり合わせ
  3. 概算レンジをご提示 → 要件確定後に正式見積
