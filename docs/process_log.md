## 2025-10-22

### サイト全体のレイアウト改善を実装

設計案に基づき、サイト全体のレイアウトを改善しました。

#### 実装内容

| セクション | Before | After |
|-----------|--------|-------|
| おすすめ商品 | 3列固定スライダー | **4列スライダー**（スマホ2列） |
| イベント | 3列固定スライダー | **4列スライダー**（スマホ2列） |
| 今月の新商品 | 縦長2枚グリッド構成 | **左右2カラム構成**（お店についてと同じ形式） |
| 色物薬膳茶 | 4列固定（4商品） | **5列固定**（5商品に拡充） |
| 季節のおすすめ | 4列固定 | 4列固定（維持） |

#### 実装詳細

**1. スライダーを4列表示に変更**
- `site/script.js`:
  - おすすめ商品・イベントの`slidesPerView`を3→4に変更
  - モバイルは2列表示に統一

**2. 今月の新商品を2カラムレイアウトに変更**
- `store_nico.html`: HTMLを`featured-product-container`に変更
- `site/style.css`:
  - `.featured-product-container`を追加（左右1:1.2の比率）
  - お店についてと同じスタイルを適用
  - レスポンシブ対応（モバイルは縦並び）

**3. 色物薬膳茶を5列表示に拡充**
- `store_nico.html`:
  - 「005 秋燈 - Shūtō」を追加（003 行火の前）
  - クラス名を`tea-gallery-grid`に変更
- `site/style.css`:
  - `.tea-gallery-grid`を追加（5列グリッド）
  - レスポンシブ: 992px以下で3列、480px以下で2列

#### 設計ポイント

- **動的要素は最小限**: スライダーは2箇所のみ（おすすめ商品・イベント）
- **レイアウトルール統一**:
  - 4列 = 更新系・注目商品（季節のおすすめ）
  - 5列 = 一覧系・ギャラリー（色物薬膳茶）
- **写真比率の統一**: 1:1 または 3:4 で統一
- **読み物感の強化**: 今月の新商品は静的な左右2カラム構成に変更

## 2025-10-22

### トップページのパス修正（Vercel対応）

**問題:** Vercelでトップページ（`index.html`）が正しく表示されず、画像とCSSが読み込まれない

**原因:** ローカルの絶対パスが使われていた
```html
<!-- ❌ 動作しない -->
<link rel="stylesheet" href="/Users/rin5uron/Desktop/nico_demo_pattern_b/site/style.css">
<img src="/Users/rin5uron/Desktop/nico_demo_pattern_b/site/top/nico_curry.jpg">
```

**修正内容:**
- すべてのパスを相対パスに変更
  - CSSファイル: `site/style.css`
  - 画像ファイル: `site/top/*.jpg`, `site/images/icons/*.svg`
- 修正箇所: 計7箇所（CSS 1箇所、画像 6箇所）

**修正後:**
```html
<!-- ✅ 正常動作 -->
<link rel="stylesheet" href="site/style.css">
<img src="site/top/nico_curry.jpg">
```

**影響範囲:** `index.html`のみ

### 薬膳茶セクションのコンパクト化

クライアントのフィードバックに基づき、「色物薬膳茶」セクションをコンパクトに調整。

#### 実装内容

- 上2つの画像比率を 4:3（横長）に変更
- 商品カードの余白調整:
  - パディング削減
  - 画像下の余白削減
  - フォントサイズ最適化
  - グリッドのgap調整
- レスポンシブ対応:
  - PC: 上2つが2列、下3つが3列
  - タブレット: 2列
  - スマホ: 1列
- 視覚的な区別を追加（上2つの商品カード）
- メディアクエリの不要なheight指定を削除し、スマホ表示の画像比率問題を解消

#### その他のセクション調整

- 「お店について」「今月の新商品」の画像を 4:3（横長）に設定
