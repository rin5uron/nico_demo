# ニコ酒店ページ「最新投稿」にInstagramを埋め込む手順

**埋め込みはできます。** ただし「1件の投稿のURL」が必要です（プロフィールURLでは表示されません）。

## 手順（2分）

1. **Instagram**（スマホでもPCでも可）で、表示したい投稿を開く  
   - 例：最新の1件の投稿
2. 投稿の右上の **「…」** をタップ／クリック
3. **「埋め込み」** を選ぶ
4. 表示されたコードのうち、次の形のURLをコピーする  
   `https://www.instagram.com/p/〇〇〇/`  
   （〇〇〇の部分は英数字の投稿IDです）
5. **store_nico.html** を開き、「最新投稿」のブロック内を探す
6. 次の行の **`data-instgrm-permalink="..."`** の `"..."` の中だけ、コピーしたURLに貼り替える  
   ```html
   <blockquote class="instagram-media" data-instgrm-permalink="ここに貼る" ...>
   ```
7. 保存してブラウザでページを再読み込みする

## 注意

- **プロフィールURL**（`https://www.instagram.com/nicosaketen/`）を入れると表示されません。必ず **1件の投稿のURL**（`/p/〇〇〇/` が含まれるもの）を入れてください。
- 最新の投稿に差し替えたいときは、同じ手順で新しい投稿のURLに貼り替えるだけです。
