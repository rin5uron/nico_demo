## 2. 提案：デザインだけ揃える場合の候補

**文言はそのまま**で、見た目（中央／右寄せ、ボタン／テキストリンク）だけを揃える案です。

### 案A：すべて「右寄せテキストリンク」に揃える

- **お知らせ**：ニコ酒店だけ、ボタン→右寄せテキストリンクに変更（文言は「ニコ酒店のInstagramで最新情報を見る」のまま）
- **イベント**：3ページとも、2行→1行の右寄せテキストリンクに変更（文言は各ページの意図のまま。例：ニコ「最新情報はInstagramをご確認ください。」＋リンク「Instagramへ」を、1行リンクにする場合はどちらの文言にするか要確認）

→ バー・カレーに合わせて、全体的に「右寄せ・テキストリンク」で統一するイメージです。

### 案B：すべて「中央・ボタン風」に揃える

- **お知らせ**：バー・カレーを、右寄せテキスト→中央のボタン風に変更（文言は現状のまま）
- **イベント**：3ページとも、2行のまま中央配置でボタン風リンクを追加するなど、ニコ酒店のお知らせに近いトーンに

→ ニコ酒店に合わせて、全体的に「中央・ボタン」で統一するイメージです。

### 案C：デザインは変えず、配置だけ微調整

- お知らせ・イベントとも、今の「ニコ＝ボタン／バー・カレー＝右寄せ」を維持
- 余白やフォントサイズだけ揃えて、バランスを取る
